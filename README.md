# Розробка веб-сайту для майстерні машинної вишивки "STITCH" з використанням сучасних технологій фронтенд-розробки
Даний репозиторій створено для зберігання вихідного коду та документації кваліфікаційної (дипломної) роботи студента спеціальності 121 "Інженерія програмного забезпечення"
## Завдання до дипломної практики
1. Сформулювати актуальність теми проєкту, завдання, мету, предмет, об’єкт 
дослідження. 
2. Виконати постановку завдання. 
3. Визначити склад АІС та розробити для неї схему бази даних, прототип 
інтерфейсу, діаграму прецедентів. 
4. Створити програмний продукт згідно постановки завдання, провести тестування. 
Частину програмного коду з коментарями навести в додатках. 
5. Розробити проект інструкції по експлуатації ІС.  
6. Зробити висновки по результатам проєктування.  
7. Скласти список використаних джерел  по проєкту.  
8. Оформити пояснювальну записку згідно вимог стандартів і методичних 
рекомендацій.  
9. Підготувати презентацію проєкту та доповідь для захисту проєкту.
